# 🏦 نظام الخزنة الاحترافي - Professional Treasury Management System

نظام إدارة خزنة شامل ومتطور يوفر جميع الأدوات اللازمة لإدارة الشؤون المالية للشركات والمؤسسات.

## ✨ الميزات الرئيسية

### 🎯 لوحة التحكم الذكية
- **إحصائيات فورية**: عرض الإيرادات، المصروفات، صافي الربح، والرصيد النقدي
- **رسوم بيانية تفاعلية**: تتبع التدفق النقدي والاتجاهات المالية
- **مؤشرات الأداء**: KPIs مهمة لاتخاذ القرارات
- **نظرة عامة شاملة**: جميع العمليات المهمة في مكان واحد

### 💰 إدارة الخزنة المالية
- **خزائن متعددة**: إدارة عدة خزائن بعملات مختلفة
- **معاملات نقدية**: تسجيل الإيرادات والمصروفات بسهولة
- **تتبع التدفق النقدي**: مراقبة دقيقة للتدفقات المالية
- **تقارير مالية**: تقارير مفصلة وشاملة

### 👥 إدارة العملاء والموردين
- **قاعدة بيانات شاملة**: معلومات مفصلة للعملاء والموردين
- **تتبع المعاملات**: ربط جميع المعاملات بالعملاء والموردين
- **تقارير مخصصة**: تقارير خاصة بكل عميل أو مورد

### 📋 نظام الفواتير المتطور
- **إنشاء فواتير احترافية**: تصميم فواتير جذابة ومهنية
- **تتبع حالة الدفع**: مراقبة الفواتير المعلقة والمدفوعة
- **تنبيهات تلقائية**: إشعارات للفواتير المتأخرة
- **حساب الضرائب**: حساب ضريبة القيمة المضافة تلقائياً

### 📦 إدارة المخزون الذكية
- **إدارة المنتجات**: إضافة وتعديل المنتجات والخدمات
- **تتبع الكميات**: مراقبة المخزون المتاح
- **تنبيهات المخزون**: تحذيرات للمخزون المنخفض
- **حركة المخزون**: تتبع جميع حركات المخزون

### 📊 التقارير المالية المتقدمة
- **تقرير الأرباح والخسائر**: P&L Statement شامل
- **تقرير التدفق النقدي**: Cash Flow Statement مفصل
- **تقارير المبيعات**: تحليل شامل للمبيعات
- **تقارير المصروفات**: تتبع وتصنيف المصروفات

### 🔔 نظام الإشعارات الذكي
- **تنبيهات المخزون**: تحذيرات عند انخفاض المخزون
- **تنبيهات الفواتير**: إشعارات للفواتير المتأخرة
- **إشعارات المعاملات**: تنبيهات للمعاملات المهمة
- **نظام إشعارات متقدم**: إدارة شاملة للإشعارات

## 🚀 التقنيات المستخدمة

### Backend
- **Flask**: إطار عمل Python خفيف وقوي
- **SQLite**: قاعدة بيانات سريعة وموثوقة
- **RESTful API**: واجهات برمجية حديثة ومنظمة
- **JWT Authentication**: نظام مصادقة آمن

### Frontend
- **HTML5 & CSS3**: معايير ويب حديثة
- **JavaScript ES6+**: برمجة حديثة ومتطورة
- **Chart.js**: رسوم بيانية تفاعلية
- **Font Awesome**: أيقونات احترافية

### Design
- **Responsive Design**: متجاوب مع جميع الأجهزة
- **Modern UI/UX**: واجهة مستخدم حديثة وجذابة
- **RTL Support**: دعم كامل للغة العربية
- **Dark/Light Theme**: ثيمات متعددة

## 🔧 التثبيت والتشغيل

### المتطلبات
- Python 3.7+
- pip (مدير حزم Python)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/your-username/treasury-system.git
cd treasury-system
```

2. **إنشاء بيئة افتراضية**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# أو
venv\Scripts\activate  # Windows
```

3. **تثبيت المتطلبات**
```bash
pip install -r requirements.txt
```

4. **تشغيل التطبيق**
```bash
python app.py
```

5. **فتح المتصفح**
```
http://localhost:5000
```

### بيانات تسجيل الدخول الافتراضية
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `password`
