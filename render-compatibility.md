# تحديثات Render Blueprint - التوافق مع الإصدارات الجديدة

## 🔄 التحديثات المطلوبة

### تغيير الخطط:
- **قبل**: `plan: starter` (لم تعد مدعومة)
- **بعد**: `plan: free` (الخطة الجديدة المجانية)

### الخدمات المحدثة:

#### 1. قاعدة البيانات:
```yaml
databases:
  - name: treasury-postgres
    plan: free  # بدلاً من starter
    databaseName: treasury_db
    user: treasury_user
```

#### 2. الخادم الخلفي:
```yaml
services:
  - type: web
    name: treasury-backend
    env: node
    plan: free  # بدلاً من starter
```

#### 3. الواجهة الأمامية:
```yaml
  - type: web
    name: treasury-frontend
    env: static
    plan: free  # إضافة الخطة
```

## 📋 قائمة الخطط المتاحة:

### الخطط المجانية:
- `free` - للاستخدام الشخصي والتطوير
- `starter` - لم تعد مدعومة ❌

### الخطط المدفوعة:
- `standard` - للاستخدام التجاري
- `pro` - للاستخدام المتقدم
- `enterprise` - للشركات الكبيرة

## 🚀 خطوات التحديث:

### 1. تحديث الملفات:
```bash
# تم تحديث render.yaml بالفعل
git add render.yaml
git commit -m "Update Render Blueprint to use free plan"
git push origin main
```

### 2. إعادة النشر:
- اذهب إلى Render Dashboard
- احذف الخدمات القديمة (إذا كانت موجودة)
- أنشئ Blueprint جديد باستخدام الملف المحدث

### 3. التحقق من النشر:
- تأكد من أن جميع الخدمات تعمل
- تحقق من Health Check
- اختبر الوظائف الأساسية

## ⚠️ ملاحظات مهمة:

### حدود الخطة المجانية:
- **الخادم الخلفي**: 750 ساعة/شهر
- **قاعدة البيانات**: 1GB تخزين
- **الواجهة الأمامية**: غير محدود
- **النوم**: الخدمات تنام بعد 15 دقيقة من عدم الاستخدام

### نصائح للتحسين:
1. **استخدم Health Checks** لضمان استمرارية الخدمة
2. **راقب الاستخدام** لتجنب تجاوز الحدود
3. **فكر في الترقية** للاستخدام الإنتاجي

## 🔧 استكشاف الأخطاء:

### مشاكل شائعة:
1. **خطأ الخطة**: تأكد من استخدام `plan: free`
2. **فشل البناء**: تحقق من `package.json`
3. **مشاكل قاعدة البيانات**: تأكد من `DATABASE_URL`

### سجلات الأخطاء:
```bash
# في Render Dashboard
Services → treasury-backend → Logs
```
