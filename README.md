# 🏦 نظام الخزنة الاحترافي المتكامل
# Professional Treasury Management System

نظام إدارة خزنة احترافي ومتطور يوفر حلول شاملة لإدارة الخزائن والمعاملات المالية مع واجهة مستخدم عصرية وميزات متقدمة.

## ✨ المميزات الرئيسية

### 🏗️ نظام هرمي للخزائن
- **الخزينة الرئيسية**: عرض إجمالي رصيد جميع الخزائن الفرعية
- **الخزائن الفرعية**: إدارة المعاملات النقدية (إيرادات ومصروفات)
- **نقل الأموال**: إمكانية نقل الأموال بين الخزائن المختلفة

### 👥 إدارة المستخدمين والأدوار
- **نظام أدوار متقدم**: مدير، مشرف، مستخدم، مراقب
- **صلاحيات مخصصة**: تحكم دقيق في الصلاحيات
- **تسجيل الأنشطة**: تتبع جميع العمليات والأنشطة

### 💰 إدارة المعاملات المالية
- **المعاملات النقدية**: إيرادات ومصروفات ونقلات
- **ربط العملاء والموردين**: ربط المعاملات بالعملاء والموردين
- **مرجع المعاملات**: نظام مراجع فريد لكل معاملة
- **تحديث تلقائي**: تحديث أرصدة الخزائن تلقائياً

### 👨‍💼 إدارة العملاء والموردين
- **قاعدة بيانات شاملة**: معلومات تفصيلية للعملاء والموردين
- **حدود الائتمان**: إدارة حدود الائتمان
- **تتبع الديون**: حساب المبالغ المستحقة
- **إحصائيات مفصلة**: عدد الفواتير والمبالغ المستحقة

### 📦 إدارة المنتجات والمخزون
- **كود المنتج (SKU)**: نظام ترميز فريد للمنتجات
- **تصنيف المنتجات**: تنظيم المنتجات حسب الفئات
- **إدارة المخزون**: تتبع الكميات والمستويات الدنيا
- **تنبيهات المخزون**: تنبيهات عند انخفاض المخزون

### 🧾 نظام الفواتير المتكامل
- **فواتير البيع والشراء**: فواتير مبيعات ومشتريات
- **عناصر الفواتير**: تفاصيل المنتجات والكميات والأسعار
- **حالات الفواتير**: معلقة، مدفوعة، جزئية، ملغاة
- **تواريخ الاستحقاق**: إدارة تواريخ الاستحقاق

### 📊 التقارير والإحصائيات
- **لوحة تحكم تفاعلية**: إحصائيات فورية ومؤشرات الأداء
- **تقارير مالية**: تقارير الإيرادات والمصروفات
- **تقارير المخزون**: تقارير حركة المخزون
- **تقارير العملاء**: تقارير المبيعات والديون

### 🔔 نظام الإشعارات
- **إشعارات فورية**: تنبيهات للأحداث المهمة
- **أنواع الإشعارات**: معلومات، تحذيرات، أخطاء، نجاح
- **تخصيص الإشعارات**: إشعارات مخصصة لكل مستخدم

### 🛡️ الأمان والحماية
- **تشفير كلمات المرور**: تشفير آمن لكلمات المرور
- **جلسات آمنة**: إدارة آمنة لجلسات المستخدمين
- **سجل الأنشطة**: تتبع جميع العمليات
- **حماية من SQL Injection**: حماية شاملة من الهجمات

## 🚀 التثبيت والتشغيل

### المتطلبات
- Python 3.7+
- pip
- SQLite3

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd treasury-professional
```

2. **تثبيت المتطلبات**
```bash
pip install -r requirements.txt
```

3. **تشغيل التطبيق**
```bash
python app.py
```

4. **فتح المتصفح**
```
http://localhost:5000
```

### بيانات تسجيل الدخول الافتراضية
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `password`

## 🌐 النشر على Render

### خطوات النشر

1. **رفع المشروع على GitHub**

2. **إنشاء خدمة ويب جديدة على Render**
   - اذهب إلى [render.com](https://render.com)
   - أنشئ حساب جديد
   - اختر "New Web Service"
   - اربط مستودع GitHub

3. **إعدادات الخدمة**
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python app.py`

4. **النشر**
   - اضغط "Create Web Service"
   - انتظر حتى يكتمل النشر

## 🗄️ قاعدة البيانات

### الجداول الرئيسية
- **users**: المستخدمون والأدوار
- **treasuries**: الخزائن الهرمية
- **customers**: العملاء
- **suppliers**: الموردين
- **products**: المنتجات
- **transactions**: المعاملات المالية
- **invoices**: الفواتير
- **invoice_items**: عناصر الفواتير
- **inventory_transactions**: حركات المخزون
- **checks**: الشيكات
- **notifications**: الإشعارات
- **system_settings**: إعدادات النظام
- **activity_logs**: سجل الأنشطة

### العلاقات
- كل معاملة مرتبطة بخزينة فرعية ومستخدم
- كل فاتورة مرتبطة بعميل أو مورد
- كل عنصر فاتورة مرتبط بمنتج
- كل حركة مخزون مرتبطة بمنتج ومستخدم
