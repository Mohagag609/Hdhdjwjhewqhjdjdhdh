# 🏦 نظام الخزنة الاحترافي
## Professional Treasury Management System

نظام خزنة احترافي شامل لإدارة العمليات المالية والنقدية مع قاعدة بيانات حقيقية.

---

## ✨ المميزات الرئيسية

### 💰 إدارة النقدية
- **سندات القبض والصرف**: تسجيل وإدارة جميع المعاملات النقدية
- **تتبع الأرصدة**: مراقبة أرصدة الخزائن المختلفة
- **أرقام معاملات فريدة**: نظام ترقيم تلقائي للمعاملات

### 🧾 إدارة الشيكات
- **شيكات القبض والدفع**: إدارة شاملة للشيكات
- **تذكير الاستحقاق**: تنبيهات للشيكات المستحقة
- **تتبع الحالة**: متابعة حالة الشيكات (معلق، مسدد، مرتد)

### 🏛️ إدارة الخزائن
- **خزائن متعددة**: إدارة أكثر من خزينة
- **تحويلات بين الخزائن**: إمكانية التحويل بين الخزائن المختلفة
- **أرصدة فورية**: عرض الأرصدة الحالية لكل خزينة

### 📊 التقارير المالية
- **تقارير يومية وشهرية**: تقارير مفصلة للعمليات المالية
- **إحصائيات شاملة**: تحليل الأداء المالي
- **تصدير البيانات**: إمكانية تصدير التقارير

### ⚙️ الإعدادات والأمان
- **نظام مستخدمين**: إدارة المستخدمين والصلاحيات
- **إعدادات قابلة للتخصيص**: تخصيص النظام حسب الحاجة
- **نسخ احتياطي**: حماية البيانات المهمة

## 🚀 طريقة التشغيل

### المتطلبات
- Python 3.7 أو أحدث
- متصفح ويب حديث

### التشغيل السريع
```bash
# تشغيل النظام
python run.py
```

### التشغيل اليدوي
```bash
# 1. تثبيت المكتبات المطلوبة
pip install -r requirements.txt

# 2. تشغيل التطبيق
python app.py
```

### الوصول للنظام
- افتح المتصفح واذهب إلى: `http://localhost:5000`
- اسم المستخدم الافتراضي: `admin`
- كلمة المرور الافتراضي: `password`

## 🗄️ قاعدة البيانات

النظام يستخدم SQLite كقاعدة بيانات مع الجداول التالية:

### الجداول الرئيسية
- **users**: المستخدمون والصلاحيات
- **treasuries**: الخزائن والأرصدة
- **cash_transactions**: المعاملات النقدية
- **checks**: الشيكات وأوراق القبض والدفع
- **parties**: العملاء والموردين
- **system_settings**: إعدادات النظام

### المميزات المتقدمة
- **Triggers تلقائية**: تحديث الأرصدة تلقائياً
- **Views للتقارير**: عرض البيانات المحسوبة
- **فهارس محسنة**: أداء سريع للاستعلامات

## 🔧 التطوير والتخصيص

### هيكل المشروع
```
treasury-system/
├── app.py                 # التطبيق الرئيسي
├── run.py                 # ملف التشغيل
├── requirements.txt       # المكتبات المطلوبة
├── database.sql          # هيكل قاعدة البيانات
├── templates/
│   └── index.html        # واجهة المستخدم
├── static/
│   └── script.js         # JavaScript
└── database/
    └── treasury.db       # قاعدة البيانات
```

### API Endpoints
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/dashboard/stats` - إحصائيات لوحة التحكم
- `GET /api/cash/transactions` - المعاملات النقدية
- `POST /api/cash/transactions` - إضافة معاملة نقدية
- `GET /api/checks` - الشيكات
- `POST /api/checks` - إضافة شيك
- `GET /api/treasuries` - الخزائن
- `GET /api/reports/daily` - التقارير اليومية
