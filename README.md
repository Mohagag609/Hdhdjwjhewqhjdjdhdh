# نظام الخزنة الاحترافي المبسط
## Professional Treasury Management System (Simplified)

نظام إدارة خزائن مالي مبسط ومركز على إدارة الخزائن والمعاملات المالية فقط.

### الميزات المتاحة

#### 1. إدارة الخزائن
- ✅ إضافة خزائن جديدة
- ✅ تعديل بيانات الخزائن
- ✅ حذف الخزائن (مع التحقق من وجود معاملات)
- ✅ عرض أرصدة الخزائن
- ✅ نظام خزائن هرمي (خزائن رئيسية وفرعية)

#### 2. المعاملات المالية
- ✅ إضافة معاملات (إيرادات، مصروفات، تحويلات)
- ✅ تعديل المعاملات
- ✅ حذف المعاملات
- ✅ عرض تاريخ المعاملات
- ✅ تحديث تلقائي لأرصدة الخزائن

#### 3. التقارير والإحصائيات
- ✅ تقرير الأرصدة
- ✅ تقرير الإيرادات
- ✅ تقرير المصروفات
- ✅ تقرير المعاملات
- ✅ إحصائيات شاملة

### التثبيت والتشغيل

#### 1. تثبيت المتطلبات
```bash
pip3 install -r requirements.txt --break-system-packages
```

#### 2. تشغيل النظام
```bash
python3 app.py
```

#### 3. الوصول للنظام
- افتح المتصفح واذهب إلى: `http://localhost:5000`
- استخدم بيانات الدخول الافتراضية:
  - **اسم المستخدم:** `admin`
  - **كلمة المرور:** `admin`

### كيفية الاستخدام

#### إضافة خزينة جديدة
1. اذهب إلى صفحة "إدارة الخزائن"
2. اضغط على "إضافة خزينة"
3. املأ البيانات المطلوبة:
   - اسم الخزينة (مطلوب)
   - الخزينة الرئيسية (اختياري)
   - الرصيد الابتدائي
   - العملة
   - الوصف
4. اضغط "إضافة الخزينة"

#### إضافة معاملة مالية
1. اذهب إلى صفحة "المعاملات"
2. اضغط على "إضافة معاملة"
3. املأ البيانات المطلوبة:
   - نوع المعاملة (إيراد/مصروف/تحويل)
   - المبلغ
   - الخزينة
   - الوصف
   - المرجع (اختياري)
   - تاريخ المعاملة
4. اضغط "إضافة المعاملة"

#### تعديل أو حذف البيانات
- **التعديل:** اضغط على أيقونة التعديل (✏️) بجانب العنصر
- **الحذف:** اضغط على أيقونة الحذف (🗑️) بجانب العنصر

### هيكل قاعدة البيانات

#### الجداول الرئيسية:
- `users` - المستخدمين
- `treasuries` - الخزائن
- `transactions` - المعاملات المالية
- `checks` - الشيكات
- `notifications` - الإشعارات
- `system_settings` - إعدادات النظام
- `activity_logs` - سجل الأنشطة

### APIs المتاحة

#### الخزائن
- `GET /api/treasuries` - عرض جميع الخزائن
- `POST /api/treasuries` - إضافة خزينة جديدة
- `PUT /api/treasuries/{id}` - تعديل خزينة
- `DELETE /api/treasuries/{id}` - حذف خزينة

#### المعاملات
- `GET /api/transactions` - عرض جميع المعاملات
- `POST /api/transactions` - إضافة معاملة جديدة
- `PUT /api/transactions/{id}` - تعديل معاملة
- `DELETE /api/transactions/{id}` - حذف معاملة

#### الإحصائيات
- `GET /api/dashboard/stats` - إحصائيات لوحة التحكم

### الأمان
- تشفير كلمات المرور
- نظام جلسات آمن
- تسجيل جميع الأنشطة
- التحقق من الصلاحيات

### المتطلبات التقنية
- Python 3.7+
- Flask 3.1.2
- SQLite3
- Flask-CORS 6.0.1

### ملاحظات مهمة
- النظام مبسط ومركز على إدارة الخزائن والمعاملات فقط
- لا يتضمن أنظمة العملاء أو المنتجات أو الفواتير
- مناسب للاستخدام في المؤسسات الصغيرة والمتوسطة
- يدعم العملة السعودية (SAR) بشكل افتراضي

### الدعم والمساعدة
في حالة وجود مشاكل أو استفسارات، يرجى مراجعة:
1. سجل الأخطاء في وحدة التحكم
2. سجل الأنشطة في قاعدة البيانات
3. التأكد من صحة البيانات المدخلة
