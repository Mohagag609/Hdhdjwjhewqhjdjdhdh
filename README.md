# نظام الخزنة الاحترافي المتكامل
## Professional Treasury Management System

نظام إدارة خزائن مالي متكامل يدعم إدارة الخزائن والمعاملات المالية والشركاء والموردين.

## الميزات الرئيسية

### 1. إدارة الخزائن الهرمية
- ✅ إضافة وتعديل وحذف الخزائن
- ✅ نظام خزائن هرمي (خزائن رئيسية وفرعية)
- ✅ كشوف حسابات هرمية
- ✅ عرض إجمالي الخزائن الفرعية

### 2. المعاملات المالية المتقدمة
- ✅ إيرادات ومصروفات
- ✅ تحويلات بين الخزائن
- ✅ ربط المعاملات بالشركاء
- ✅ تصنيف المعاملات حسب النوع

### 3. نظام الشركاء والموردين
- ✅ إدارة شاملة للشركاء والموردين
- ✅ أنواع الشركاء: مورد، عميل، مورد وعميل
- ✅ كشوف حسابات مفصلة لكل شريك
- ✅ تتبع الإيرادات والمصروفات لكل شريك

### 4. أنواع الإيرادات والمصروفات
- ✅ أنواع الإيرادات: مبيعات، خدمات، استثمارات، إيرادات أخرى
- ✅ أنواع المصروفات: مشتريات، رواتب، إيجار، كهرباء وماء، صيانة، مصروفات أخرى
- ✅ تصنيف تلقائي للمعاملات

### 5. التقارير والإحصائيات
- ✅ تقارير الأرصدة
- ✅ تقارير الإيرادات والمصروفات
- ✅ تقارير المعاملات
- ✅ كشوف الحسابات الهرمية
- ✅ إحصائيات شاملة

## التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
pip3 install -r requirements.txt --break-system-packages
```

### 2. تشغيل النظام
```bash
python3 app.py
```

### 3. الوصول للنظام
- افتح المتصفح واذهب إلى: `http://localhost:5000`
- استخدم بيانات الدخول الافتراضية:
  - **اسم المستخدم:** `admin`
  - **كلمة المرور:** `admin`

## دليل الاستخدام

### إدارة الخزائن

#### إضافة خزينة جديدة
1. اذهب إلى "إدارة الخزائن"
2. اضغط "إضافة خزينة"
3. املأ البيانات:
   - اسم الخزينة (مطلوب)
   - الخزينة الرئيسية (اختياري)
   - الرصيد الابتدائي
   - العملة
   - الوصف
4. اضغط "إضافة الخزينة"

#### كشف الحساب الهرمي
- اضغط على أيقونة كشف الحساب بجانب الخزينة
- يعرض الخزينة الرئيسية + جميع الخزائن الفرعية
- إجمالي شامل لجميع الأرصدة

### المعاملات المالية

#### إضافة معاملة جديدة
1. اذهب إلى "المعاملات"
2. اضغط "إضافة معاملة"
3. اختر نوع المعاملة:
   - **إيراد:** اختر نوع الإيراد
   - **مصروف:** اختر نوع المصروف
   - **تحويل:** اختر الخزينة المستقبلة
4. املأ البيانات المطلوبة
5. اضغط "إضافة المعاملة"

#### أنواع المعاملات
- **الإيرادات:** تزيد رصيد الخزينة
- **المصروفات:** تقلل رصيد الخزينة
- **التحويلات:** تنقل المال بين الخزائن

### إدارة الشركاء والموردين

#### إضافة شريك/مورد جديد
1. اذهب إلى "الشركاء والموردين"
2. اضغط "إضافة شريك/مورد"
3. املأ البيانات:
   - اسم الشريك/المورد (مطلوب)
   - نوع الشريك (مورد/عميل/كلاهما)
   - البريد الإلكتروني
   - رقم الهاتف
   - العنوان
   - حد الائتمان
4. اضغط "إضافة الشريك"

#### كشف حساب الشريك
- اضغط على أيقونة كشف الحساب بجانب الشريك
- يعرض جميع المعاملات المرتبطة بالشريك
- إجمالي الإيرادات والمصروفات

## APIs المتاحة

### الخزائن
- `GET /api/treasuries` - عرض جميع الخزائن
- `POST /api/treasuries` - إضافة خزينة جديدة
- `PUT /api/treasuries/{id}` - تعديل خزينة
- `DELETE /api/treasuries/{id}` - حذف خزينة
- `GET /api/treasuries/{id}/hierarchical-statement` - كشف حساب هرمي

### المعاملات
- `GET /api/transactions` - عرض جميع المعاملات
- `POST /api/transactions` - إضافة معاملة جديدة
- `PUT /api/transactions/{id}` - تعديل معاملة
- `DELETE /api/transactions/{id}` - حذف معاملة

### الشركاء
- `GET /api/partners` - عرض جميع الشركاء
- `POST /api/partners` - إضافة شريك جديد
- `GET /api/partners/{id}/statement` - كشف حساب الشريك

### أنواع الإيرادات والمصروفات
- `GET /api/income-types` - أنواع الإيرادات
- `GET /api/expense-types` - أنواع المصروفات

### الإحصائيات
- `GET /api/dashboard/stats` - إحصائيات لوحة التحكم

## هيكل قاعدة البيانات

### الجداول الرئيسية:
- `users` - المستخدمين
- `treasuries` - الخزائن
- `transactions` - المعاملات المالية
- `partners` - الشركاء والموردين
- `income_types` - أنواع الإيرادات
- `expense_types` - أنواع المصروفات
- `checks` - الشيكات
- `notifications` - الإشعارات
- `system_settings` - إعدادات النظام
- `activity_logs` - سجل الأنشطة
