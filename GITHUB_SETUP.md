# 🚀 إرشادات رفع المشروع على GitHub

## خطوات رفع نظام الخزنة الاحترافي على GitHub:

### 1. إنشاء مستودع جديد على GitHub
- اذهب إلى: https://github.com/new
- **اسم المستودع:** `professional-treasury-system`
- **الوصف:** `نظام الخزنة الاحترافي المتكامل - Professional Treasury Management System`
- **نوع المستودع:** Public أو Private (حسب اختيارك)
- **لا تضع:** README, .gitignore, أو License (لأننا لدينا ملفات موجودة)

### 2. ربط المستودع بالمشروع المحلي
بعد إنشاء المستودع، انسخ رابط المستودع واتبع الأوامر التالية:

```bash
# اربط المستودع المحلي بالمستودع على GitHub
git remote add origin https://github.com/USERNAME/professional-treasury-system.git

# ارفع الفرع الحالي
git push -u origin professional-treasury-system
```

### 3. التحقق من الرفع
```bash
# تحقق من حالة المستودع
git status

# تحقق من الفروع
git branch -a

# تحقق من الـ remotes
git remote -v
```
