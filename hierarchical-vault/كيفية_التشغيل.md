# 🚀 كيفية تشغيل المشروع على Render

## الخطوات الأساسية:

### 1️⃣ إعداد قاعدة البيانات Neon
1. اذهب إلى [neon.tech](https://neon.tech)
2. أنشئ حساب جديد
3. أنشئ مشروع جديد
4. احصل على معلومات الاتصال (Host, Database, Username, Password)
5. شغل ملف `database/schema-neon.sql` في Neon Console

### 2️⃣ رفع المشروع إلى GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/hierarchical-vault.git
git push -u origin main
```

### 3️⃣ النشر على Render
1. اذهب إلى [render.com](https://render.com)
2. اضغط "New +" → "Blueprint"
3. اختر المستودع
4. حدث متغيرات البيئة في Backend:
   - `DB_HOST` = معلومات Neon Host
   - `DB_NAME` = اسم قاعدة البيانات
   - `DB_USER` = اسم المستخدم
   - `DB_PASSWORD` = كلمة المرور
   - `DB_SSL` = true

5. حدث متغيرات البيئة في Frontend:
   - `VITE_API_URL` = https://hierarchical-vault-backend.onrender.com/api

### 4️⃣ تشغيل المشروع
1. اضغط "Manual Deploy" في Backend
2. اضغط "Manual Deploy" في Frontend
3. انتظر اكتمال البناء

## ✅ النتيجة:
- Backend: `https://hierarchical-vault-backend.onrender.com`
- Frontend: `https://hierarchical-vault-frontend.onrender.com`

## 🔧 إذا واجهت مشاكل:
1. تحقق من متغيرات البيئة
2. راجع logs في Render
3. تأكد من صحة معلومات Neon
