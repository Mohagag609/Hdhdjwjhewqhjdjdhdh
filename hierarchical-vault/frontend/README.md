# خزينة هرمية - الواجهة الأمامية

## الوصف
واجهة مستخدم تفاعلية لنظام إدارة الخزائن الهرمية، مبني باستخدام React و Vite.

## المتطلبات
- Node.js 16+
- npm أو yarn

## التثبيت

1. تثبيت التبعيات:
```bash
npm install
```

2. إعداد متغيرات البيئة:
```bash
# تأكد من وجود ملف .env مع الإعدادات التالية:
VITE_API_URL=http://localhost:5000/api
```

## التشغيل

### وضع التطوير:
```bash
npm run dev
```

### بناء الإنتاج:
```bash
npm run build
```

### معاينة الإنتاج:
```bash
npm run preview
```

## الميزات

### الصفحات الرئيسية
- **لوحة التحكم**: نظرة عامة على الإحصائيات والتقارير الأخيرة
- **الخزائن**: إدارة الخزائن الرئيسية والفرعية
- **المشاريع**: إدارة المشاريع والأنشطة المالية
- **التقارير**: إدارة التقارير المالية والإيرادات والمصروفات

### المكونات
- **Header**: شريط التنقل الرئيسي مع قائمة المستخدم
- **Layout**: تخطيط الصفحات الأساسي
- **Loading**: مؤشر التحميل
- **Error**: عرض الأخطاء
- **EmptyState**: حالات عدم وجود بيانات

### الخدمات
- **API Service**: خدمة للتفاعل مع الخادم الخلفي
- **Auth Context**: إدارة حالة المصادقة

## التقنيات المستخدمة

- **React 19**: مكتبة واجهة المستخدم
- **Vite**: أداة البناء والتطوير
- **Tailwind CSS**: إطار عمل CSS
- **React Router**: توجيه الصفحات
- **Axios**: طلبات HTTP
- **Heroicons**: مكتبة الأيقونات

## الهيكل

```
src/
├── components/          # المكونات القابلة لإعادة الاستخدام
├── pages/              # صفحات التطبيق
├── context/            # React Context
├── services/           # خدمات API
├── utils/              # أدوات مساعدة
├── hooks/              # React Hooks مخصصة
├── App.jsx             # المكون الرئيسي
└── main.jsx            # نقطة الدخول
```

## الأمان

- حماية الصفحات من الوصول غير المصرح به
- إدارة التوكنات تلقائياً
- إعادة التوجيه عند انتهاء صلاحية التوكن
