# البدء السريع - خزينة هرمية

## 🚀 التشغيل السريع

### الطريقة الأولى: استخدام سكريبت التشغيل
```bash
# تشغيل السكريبت التلقائي
./start.sh
```

### الطريقة الثانية: التشغيل اليدوي

#### 1. إعداد قاعدة البيانات
```bash
# الدخول إلى MySQL
mysql -u root -p

# تشغيل ملف إنشاء قاعدة البيانات
source database/schema.sql
```

#### 2. تشغيل الخادم الخلفي
```bash
cd backend
npm install
cp .env.example .env
# تحديث ملف .env بالقيم المناسبة
npm run dev
```

#### 3. تشغيل الواجهة الأمامية
```bash
cd frontend
npm install
npm run dev
```

## 🌐 الوصول للنظام

- **الواجهة الأمامية**: http://localhost:3000
- **API الخلفي**: http://localhost:5000

## 👤 بيانات الدخول الافتراضية

- **اسم المستخدم**: admin
- **كلمة المرور**: password

## 📋 الميزات المتاحة

### للمدير (Admin):
- ✅ إدارة الخزائن (إضافة، تعديل، حذف)
- ✅ إدارة المشاريع (إضافة، تعديل، حذف)
- ✅ إدارة المستخدمين
- ✅ عرض جميع التقارير المالية
- ✅ إحصائيات شاملة

### للمستخدم العادي (User):
- ✅ عرض الخزائن والمشاريع
- ✅ إضافة التقارير المالية
- ✅ عرض التقارير الخاصة به
- ✅ تعديل الملف الشخصي

## 🔧 إعدادات مهمة

### ملف backend/.env
```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=hierarchical_vault
DB_PORT=3306
JWT_SECRET=your_super_secret_jwt_key_here
JWT_EXPIRES_IN=24h
PORT=5000
NODE_ENV=development
CORS_ORIGIN=http://localhost:3000
```

### ملف frontend/.env
```env
VITE_API_URL=http://localhost:5000/api
```

## 🎯 الاستخدام الأساسي

### 1. إنشاء خزينة جديدة
1. اذهب إلى "الخزائن"
2. اضغط "إضافة خزينة جديدة"
3. أدخل اسم الخزينة والوصف
4. اختر الخزينة الأب (اختياري)
5. اضغط "إنشاء الخزينة"

### 2. إنشاء مشروع جديد
1. اذهب إلى "المشاريع"
2. اضغط "إضافة مشروع جديد"
3. أدخل اسم المشروع والوصف
4. اختر الخزينة المرتبطة
5. حدد حالة المشروع
6. اضغط "إنشاء المشروع"

### 3. إضافة تقرير مالي
1. اذهب إلى "التقارير"
2. اضغط "إضافة تقرير جديد"
3. اختر نوع التقرير (إيراد/مصروف)
4. أدخل المبلغ والوصف
5. اختر الخزينة والمشروع
6. حدد تاريخ التقرير
7. اضغط "إنشاء التقرير"
