# 🔍 استكشاف مشكلة تسجيل الدخول

## ❌ المشكلة:
```
مش بيدخل مفيش اي حاجه بتحصل
```

## 🔧 خطوات الاستكشاف:

### 1️⃣ تحقق من API
1. اذهب إلى `https://hierarchical-vault.onrender.com/api`
2. يجب أن ترى: `{"message":"Hierarchical Vault API"}`

### 2️⃣ اختبار تسجيل الدخول
1. افتح ملف `test-api.html` في المتصفح
2. تأكد من API URL صحيح
3. جرب تسجيل الدخول بـ:
   - **Username**: `admin`
   - **Password**: `password`

### 3️⃣ تحقق من قاعدة البيانات
1. اذهب إلى [console.neon.tech](https://console.neon.tech)
2. اختر مشروعك
3. اذهب إلى "SQL Editor"
4. شغل هذا الاستعلام:

```sql
SELECT * FROM users WHERE username = 'admin';
```

### 4️⃣ تحقق من Logs في Render
1. اذهب إلى [dashboard.render.com](https://dashboard.render.com)
2. اختر خدمتك
3. اضغط "Logs"
4. ابحث عن أخطاء

## 🚨 المشاكل المحتملة:

**الحل:**

```sql
INSERT INTO users (username, email, password_hash, role) VALUES 
('admin', 'admin@example.com', '$2a$10$92IXUNpkjO0rOQ5byMi.Ye4oKoEa3Ro9llC/.og/at2.uheWG/igi', 'admin');
```

### 2. API لا يعمل
**الحل:**
- تحقق من متغيرات البيئة
- تأكد من اتصال قاعدة البيانات

### 3. Frontend لا يرسل الطلب
**الحل:**
- افتح Developer Tools (F12)
- اذهب إلى Network tab
- جرب تسجيل الدخول
- شاهد الطلبات المرسلة

### 4. CORS مشكلة
**الحل:**
- تأكد من CORS_ORIGIN صحيح
- أعد نشر Backend
