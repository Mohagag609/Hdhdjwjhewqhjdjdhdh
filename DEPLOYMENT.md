# 🚀 دليل النشر - نظام الخزنة الاحترافي المتكامل

## النشر على Render

### المتطلبات
- حساب على موقع Render.com
- مشروع مرفوع على GitHub

### خطوات النشر

#### 1. رفع المشروع على GitHub
```bash
git add .
git commit -m "Professional Treasury Management System"
git push origin main
```

#### 2. إنشاء خدمة جديدة على Render

1. اذهب إلى [render.com](https://render.com)
2. سجل الدخول أو أنشئ حساب جديد
3. اضغط على "New +" ثم اختر "Web Service"
4. اختر "Build and deploy from a Git repository"
5. اربط حساب GitHub الخاص بك
6. اختر المستودع الذي يحتوي على المشروع

#### 3. إعدادات الخدمة

**اسم الخدمة:** `treasury-professional`

**البيئة:** `Python 3`

**الفرع:** `main`

**أمر البناء (Build Command):**
```bash
pip install -r requirements.txt
```

**أمر التشغيل (Start Command):**
```bash
python app.py
```

#### 4. متغيرات البيئة (Environment Variables)

لا تحتاج إلى متغيرات بيئة إضافية، التطبيق سيعمل بالضبط كما هو.

#### 5. بيانات تسجيل الدخول الافتراضية

- **اسم المستخدم:** `admin`
- **كلمة المرور:** `password`

#### 6. النشر

1. اضغط على "Create Web Service"
2. انتظر حتى يكتمل البناء والنشر (5-10 دقائق)
3. ستحصل على رابط للتطبيق مثل: `https://treasury-professional.onrender.com`
